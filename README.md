# Social Network API

## Description

This is an API for a social network web application where users can share their thoughts, react to friendsâ€™ thoughts, and create a friend list. Users can interact with each other by creating post and adding friends. Those friends can then display reactions to the user's post.


## Usage

To use the application, open the application in a program like insomnia. Then run the post, put, and delete commands to interact with the web application.

## Installation

1. Clone the repo
   ```sh
   git clone https://github.com/hoeferg/Socail-API-Network
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start the application
   ```sh
   npm start
